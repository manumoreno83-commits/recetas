# Fotos Familiares para el Carrusel

Esta carpeta debe contener **6 fotos familiares** que se mostrarán en el carrusel de la página principal.

## Instrucciones:

1. Añade 6 fotos de familia a esta carpeta
2. Nómbralas exactamente así:
   - `foto1.jpg`
   - `foto2.jpg`
   - `foto3.jpg`
   - `foto4.jpg`
   - `foto5.jpg`
   - `foto6.jpg`

## Recomendaciones:

- **Formato**: JPG o JPEG preferiblemente
- **Tamaño recomendado**: 1200x800px o similar (horizontal)
- **Peso**: Máximo 500KB por foto (para carga rápida)
- **Contenido**: Fotos de reuniones familiares, comidas, celebraciones, etc.

## Nota:

Si no tienes 6 fotos, puedes usar la misma foto varias veces con nombres diferentes, o crear fotos placeholder temporales hasta que tengas las definitivas.

Una vez añadidas las fotos, el carrusel se mostrará automáticamente en la página principal.
